# Web Security Labs & Automation

This repository contains my personal work and research in web application security, focused on understanding, exploiting, and mitigating common vulnerabilities.

The goal is not only to demonstrate offensive capabilities, but also to provide a structured and professional approach to:

- Vulnerability discovery
- Exploitation techniques
- Automation of attack patterns
- Detection strategies
- Mitigation and secure design

## Current Scope

At the moment, the repository focuses on:

- SQL Injection (SQLi)

More vulnerabilities will be added progressively (XSS, CSRF, SSRF, etc.).

## Repository Structure

Each vulnerability type includes:

- Real lab write-ups (e.g., PortSwigger)
- Custom automation scripts (Python)
- Payload collections
- Backend assumptions and analysis
- Detection techniques
- Mitigation strategies

## Disclaimer

All content in this repository is for educational purposes only and has been developed in controlled environments such as labs and intentionally vulnerable applications.

## Technologies Used

- Python
- Burp Suite
- PortSwigger Web Security Academy
- HTTP libraries (requests, urllib3)
