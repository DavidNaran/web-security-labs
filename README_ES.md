# Laboratorios y Automatización de Seguridad Web

Este repositorio recoge mi trabajo personal en seguridad de aplicaciones web, incluyendo el análisis, explotación y mitigación de vulnerabilidades.

Actualmente está enfocado en:

- SQL Injection (SQLi)

El objetivo es documentar laboratorios reales (como PortSwigger) y desarrollar automatizaciones en Python que permitan entender patrones de ataque de forma estructurada.

Cada vulnerabilidad incluye:

- Write-ups de laboratorios
- Scripts de automatización
- Payloads
- Análisis del backend
- Técnicas de detección
- Estrategias de mitigación

Todo el contenido ha sido desarrollado en entornos controlados con fines educativos.
